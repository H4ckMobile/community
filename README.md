# h4ckmobile · community

> The community space for [h4ckmobile.com](https://h4ckmobile.com) readers.

Welcome! This is where readers and I interact about the blog.

## What goes here

- 🐛 **Found a typo or technical error in a post?** [Open an issue](https://github.com/h4ckmobile/community/issues/new?template=post-correction-en.yml)
- 💡 **Have a topic suggestion?** [Suggest a post](https://github.com/h4ckmobile/community/issues/new?template=post-suggestion-en.yml)
- ❓ **Got a question about a post?** [Start a discussion](https://github.com/h4ckmobile/community/discussions/categories/q-a)
- 🔧 **Hit a bug on the site itself?** [Report it](https://github.com/h4ckmobile/community/issues/new?template=site-bug-en.yml)

## What does NOT go here

- Specific technical questions about your own pentests (use Discord/forums)
- Unauthorized vulnerability disclosures (responsible disclosure → email)
- Anything that violates the [Code of Conduct](./CODE_OF_CONDUCT.md)
